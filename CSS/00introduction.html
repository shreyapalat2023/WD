  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

Include CSS in a webpage

CSS is used for styling the look and formatting of a document written in HTML. There are three ways to add CSS in HTML

  • Inline CSS:Styles added directly to the HTML element.

    • Inline style is the approach of adding CSS rules directly to the HTML element using the style attribute.

  • Internal CSS:Styles defined at the head section of the document.

    • Internal CSS applies CSS styles to a specific HTML document. Internal CSS is defined inside an HTML document using <style> attribute within the head tag of an HTML.

  • External CSS:Styles defined in a separate file.

    • External CSS is an approach to applying CSS styles to HTML pages by defining the CSS in a separate file.

    The external CSS file should have a .css extension.

    We use <link> tag to link the css file to an HTML document.

    href - URL or file path to the external CSS file.

    rel = "stylesheet" indicates the linked document is a CSS file.

Inline Style Override Internal Style

If an internal CSS and inline CSS are both applied to a tag, the styling from the inline tag is applied.
