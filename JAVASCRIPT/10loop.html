Loop

Loops in JavaScript are used to execute a block of code repeatedly as long as a specified condition is true. There are several types of loops available in JavaScript:

Types of Loops

  • for statement
  • do while statement
  • while statement
  • labeled statement
  • break statement
  • continue statement
  • for in statement
  • for of statement

For Loop:

The for loop is used to iterate over a block of code a specified number of times.

Example 
            for (initialization; condition; iteration) {
                // Code to be executed
            }

While loop

The while loop repeats a block of code as long as a specified condition is true.

Example 
            while (condition) {
                // Code to be executed
            }

            let i = 0;
            while (i < 5) {
            console.log(i);  // Output: 0, 1, 2, 3, 4
            i++;
        }

do..while loop:

Similar to the while loop, but it always executes the block of code at least once, even if the condition is false.

Example 
                    do {
                    // Code to be executed
                    } while (condition);
                
                    let i = 0;
                    do {
                    console.log(i);  // Output: 0
                    i++;
                    } while (i < 0);

for...in Loop:

The for...in loop iterates over the enumerable properties of an object.

Example 
            for (variable in object) {
                // Code to be executed
            }
        
            const obj = { a: 1, b: 2, c: 3 };
            for (let key in obj) {
                console.log(key);  // Output: a, b, c
                console.log(obj[key]);  // Output: 1, 2, 3
            }

for...of Loop:

The for...of loop iterates over the values of an iterable object (e.g., arrays, strings).

Example 
            for (variable of iterable) {
                // Code to be executed
            }
        
            const arr = [1, 2, 3];
            for (let value of arr) {
                console.log(value);  // Output: 1, 2, 3
            }

Question Answer

Entry controlled loop and exit controlled loop

Entry control and exit control loops are concepts used in programming to describe different types of loops based on when the loop condition is evaluated.

Entry Control Loop:

  • In an entry control loop, the loop condition is evaluated before the loop body is executed. If the condition evaluates to true, the loop body is executed; otherwise, the loop terminates without executing the body.
  • Examples of entry control loops include the "while" loop and the "for" loop in most programming languages.
  • Entry control loops are suitable when the loop body may not need to execute at all if the condition is false from the beginning.
  • Example

Exit Control Loop:

  • In an exit control loop, the loop body is executed at least once before the loop condition is evaluated. After executing the loop body, the condition is checked. If it evaluates to true, the loop continues; otherwise, it terminates.
  • The "do-while" loop is an example of an exit control loop.
  • Exit control loops are suitable when you want the loop body to execute at least once, regardless of the initial condition
