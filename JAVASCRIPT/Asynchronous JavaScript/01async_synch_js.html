Synchronous and Asynchronous JavaScript

Synchronous:

Synchronous means the code runs in a particular sequence of instructions given in the programm. Each instruction waits for the previous instructoin to completer its execution.

Asynchronous:

Due to synchronous programming, somethimes imp instructions get blocked due to some previous instructions, which causes a delay in the UI.

Asynchronous code execution allows to execute next instructions immediately and doesn't block the flow.
