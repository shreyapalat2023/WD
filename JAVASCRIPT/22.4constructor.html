JavaScript Class constructor

The constructor() method is a special method for creating and initializing objects created within a class.

The constructor() method is called automatically when a class is initiated, and it has to have the exact name "constructor", in fact, if you do not have a constructor method, JavaScript will add an invisible and empty constructor method.

When we want to do something when initializing object we can use contructor.

Note:A class cannot have more than one constructor() method. This will throw a SyntaxError.

We cannot return anything in constructor.

There is no requirement of calling constructor.

Syntax:

        class Person {
            construcor(name,age){
                this.name = name;
                this.age = age;
            }
        }

        const person1 = new Person("Shreya",24);
        console.log(person1.name,person1.age) //Output: Shreya 24

You can also have a default constructor if you don't need to pass any initial values:

Syntax:

        class Animal {
            construcor() {
                this.type = 'Unknown';
            }

            identify() {
                console.log(`This is a ${this.type}.`);
            }
        }
        const animal1 = Animal();
        animal1.identify(); //Outpur: This is a Unknown.
