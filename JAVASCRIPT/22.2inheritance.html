Inheritance(Reusability)

Definition

ðŸ’¬ "Inheritance allows one class to inherit properties and methods from another, reducing code duplication."

ðŸ‘‰ Example (Animal and Dog)

"Consider an 'Animal' class with a 'speak()' method. A 'Dog' class can inherit from 'Animal' and override the method to make a barking sound."

ðŸ”¹ Code Reference (Verbal Explanation)

"In JavaScript, we use extends and super(). The Dog class inherits from Animal, meaning it gets all Animal properties but can customize the speak() method."

  1. 'extends' Keyword:

    Used to create a class that is a child of another class.

  2. 'super' Keyword:

    Used to call the constructor of the parent class and to access parent class methods.

  3. Method Overriding:

    Child classes can override methods of the parent class to provide specific implementations.

Example

            class Animal {
                constructor(name) {
                  this.name = name;
                }
              
                speak() {
                  console.log(`${this.name} makes a noise.`);
                }
              }
              
              class Dog extends Animal {
                constructor(name, breed) {
                  super(name); // Calls the parent constructor
                  this.breed = breed;
                }
              
                speak() {
                  console.log(`${this.name} barks!`);
                }
              }
              
              const dog1 = new Dog("Buddy", "Golden Retriever");
              dog1.speak(); // Buddy barks!

Note:The Dog class extends Animal, reusing its functionality and overriding speak().
