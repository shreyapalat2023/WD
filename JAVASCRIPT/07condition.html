Conditional Statements

JavaScript Conditional statements allow you to execute specific blocks of code based on conditions. If the condition meets then a particular block of action will be executed otherwise it will execute another block of action that satisfies that particular
There are several methods that can be used to perform Conditional Statements in JavaScript.

Types of Conditional Statements:

If Statement:

The if statement is used to execute a block of code if a specified condition is true.

Example 
            if (condition) {
                // Code to be executed if condition is true
            }

if else Statement

The if...else statement is used to execute one block of code if the condition is true and another block of code if the condition is false.


        if (condition1) {
               Executes when condition1 is true
        if (condition2) {
                Executes when condition2 is true
          }
        }

if..else if.. Statement:

The if...else if...else statement allows you to specify multiple conditions and execute different blocks of code based on which condition is true.

        if (1st condition) {
            // Code for 1st condition
        } else if (2nd condition) {
            // code for 2nd condition
        } else if (3rd condition) {
            // Code for 3rd condition
        } else {
            //  code that will execute if all 
            // above conditions are false
        }

switch Statement:

The switch statement is used to select one of many code blocks to be executed.

           switch (expression) {
              case value1:
                // Code to be executed if expression equals value1
                break;
               case value2:
                // Code to be executed if expression equals value2
                break;
              default:
                // Code to be executed if expression doesn't match any case
            }

Ternary Operator (Conditional Operator):

The ternary operator condition ? expression1 : expression2 is a concise way to write conditional statements. It evaluates the condition and returns expression1 if the condition is true, otherwise it returns expression2. 

        var result = (condition) ? expression1 : expression2;

1.Differene between exec() and test() method

In JavaScript, both exec() and test() are methods of the RegExp (regular expression) object used for pattern matching, but they have different purposes and return values:

exec() Method:
  • The exec() method is a method of the RegExp object that searches for a match in a specified string
  • It returns an array containing information about the first match found, or null if no match is found
  • If the regular expression has the global flag (g), subsequent calls to exec() will continue to search for matches in the same string, starting from the index of the last match.
  • The returned array includes additional properties (index, input) indicating the index of the match in the string and the original input string, respectively.
test() Method:
  • The test() method is also a method of the RegExp object that searches for a match in a specified string.
  • It returns a boolean value indicating whether or not the pattern matches the string.
  • Unlike exec(), test() does not return details about the match; it simply returns true if there is a match and false otherwise.
