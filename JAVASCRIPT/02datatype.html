Data Type

Definition: In JavaScript, a datatype refers to the classification of the types of data that variables can hold. JavaScript has several built-in datatypes, which can be categorized into two main groups: primitive datatypes and non-primitive datatypes (also known as reference datatypes).

Question Answer

1. What are the various data types that exist in JavaScript?

In programming, data types categorize the kinds of data that a programming language can operate on. These can broadly be classified into two categories: primitive and non-primitive (or reference) data types.

Primitive Data Types:

Definition: Primitive data types are the most basic data types available in a programming language. They directly represent simple values and are not composed of other data types.
Characteristics:
  • They are immutable, meaning they cannot be modified
  • They have a fixed size determined by the programming language.
  • They are typically passed by value.

There are seven primitive datatypes:

  • Null: Denotes the absence of any value or object.
    Example: null
  • Number: Numeric data type, which can be integer or floating-point.
    Example: 42, 3.14, 0xFF (hexadecimal), NaN (Not-a-Number), Infinity
  • BigInt:
  • Boolean: Represents a logical entity, either true or false.
    Example: true, false
  • String: A sequence of characters, enclosed within single (''), double (" "), or backticks (``) quotes.
    Example: "Hello, World!", 'JavaScript', ` `Hello, World!` `
  • Symbol: Introduced in ECMAScript 6, represents a unique identifier. Symbols are often used as property keys in objects.
    Example: Symbol('foo')
  • Undefined: Denotes a variable that has been declared but not assigned a value.
    Example: 'undefined'

Nonprimitive datatypes:

Definition: Non-primitive data types, also known as reference types, are more complex data types that are composed of multiple primitive or other non-primitive data types.
Characteristics:
  • They can hold multiple values and have dynamic sizes
  • They are mutable, meaning their values can be changed.
  • They are typically passed by reference.

Lists of Non-primitive datatypes:

  • Object: A collection of key-value pairs, where values can be primitive types or other objects.
    Example: 
                                let person = {
                                name: 'John',
                                age: 30,
                                isStudent: false
                            };
  • Array: A special type of object used to store lists of elements.
    Example: 
                                let colors = ['red', 'green', 'blue'];
  • Function: A callable object that can be invoked to perform a particular task.
    Example: 
                                function add(a, b) {
                                return a + b;
                            }
  • Date: Represents dates and times.
    Example: 
    let today = new Date();
  • RegExp: Represents regular expressions for pattern matching.
    Example: 
    let pattern = /[a-z]+/g;
  • Map: A collection of key-value pairs where keys can be of any data type.
    Example: 
                                let myMap = new Map();
                            myMap.set('key1', 'value1');
  • Set: A collection of unique values of any data type.
    Example: 
                                let mySet = new Set([1, 2, 3, 4, 5]);
  • Promise: Represents the eventual completion (or failure) of an asynchronous operation, and its resulting value.
    Example: 
                                let promise = new Promise((resolve, reject) => {
                            // Async operation
                            if (success) {
                                resolve('Success');
                            } else {
                                reject('Error');
                            }
                        });
  • Buffer(Node.js): Used to deal with binary data directlyspan>
  • Typed Arrays(e.g, Int8Array,Uint8Array): Arrays with specific data types and fixed lengths, optimized for numerical operations.

JavaScript is a dynamically typed language, meaning that variables are not bound to a specific data type, and data types are automatically converted as needed during operations.
