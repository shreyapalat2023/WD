Practical Use Case:

Consider a scenario where you have a base class for Employee and specialized classes for Manager and Developer

This example shows how you can use inheritance to create specific employee types with additional properties and behaviors while still sharing common functionality from the base class.

Base Class vs Derived Class

  1. Base class:This class has a constructor that initialized the "name" and "id" properties and a method "getDetails".
  2. Derived Class:Here, We have Two Derived Classes Manager and Developer which inherit name and id properties of base class Employee. They both have their own constructor that calls the parent class's constructor using "super" keyword. It also add a "getDetails()" method and override the method in base class.
