Disable Weekends

Explanation:

  • dateInput.addEventListener('input', ...): This line adds an event listener to the dateInput element that listens for the input event. The input event is triggered every time the user changes the value of the input field (e.g., selects a new date).
  • function() {...}: This is an anonymous function that gets executed whenever the input event is triggered.
  • this.value:Inside the event listener function, this refers to the dateInput element. this.value gives the current value of the input field, which is the date selected by the user in the format YYYY-MM-DD.
  • new Date(this.value):This converts the selected date (which is a string) into a JavaScript Date object. This allows us to use date-specific methods on it.
  • dateValue.getDay() The getDay() method of the Date object returns the day of the week for the specified date as an integer (0 for Sunday, 1 for Monday, 2 for Tuesday, ..., 6 for Saturday).
