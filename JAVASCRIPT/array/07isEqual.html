Write a method that makes a shallow compare of two arrays and returns true if they are identical.

  • Length Check: The function first checks if the lengths of arr1 and arr2 are the same. If not, it immediately returns false because arrays of different lengths cannot be identical.
  • Element-wise Comparison: If the lengths are the same, the function then iterates through each element of the arrays. It compares corresponding elements (arr1[i] and arr2[i]) at each index.
  • Return false on Mismatch If any pair of elements does not match (arr1[i] !== arr2[i]), the function returns false, indicating that the arrays are not identical.
  • Return true if No Mismatches:If all elements match, the function returns true, indicating that the arrays are identical.
