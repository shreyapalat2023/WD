Write a method that splits an array into parts of determined size.

Explanation:

  1. Function Parameter:
    • array: The input array that you want to split into chunks.
    • size:The size of each chunk.
  2. Initialization:li>
    • const chunkedArr = []: This initializes an empty array "chinkedArr", which will store all the resulting chunks.
    • "let index= 0" : This initializes an index variable to keep track of the current position in the array as you loop through it.
  3. While Loop:
    • "While (index < array.length)" : The loop continues as long as "index" is less than the length of the array. This ensures that all elments of the array are processed.
  4. Slicing the Array
    • "ChunkedArray.push(array.slice(index + size)): It creates a subarray (or chunk) starting from the current "index" and ending at "index + size"
    • This subarray is then pushed into "chunkedArr"
  5. Incrementing the Index:
    • "index += size" After each iteration, the index is incremented by "size", so the next chunk starts from where the previous chunk ended.

For Example,

If size = 2 and the array is [1,2,3,4]

  • First iteration: "index = 0", slice from 0 to 2 resulting in [1,2]
  • Second iteration: "index = 2", slice from 2 to 4 resulting in [3,4]
