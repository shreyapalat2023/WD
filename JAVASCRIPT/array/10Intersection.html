Write a method that creates an array of unique values that are included in all given arrays.

Explanation:

  1. Start with the First Array.
    • We begin with the first array (acc initially). This is the starting point.
    • acc stands for accumulator and second parameter stands for next element of arrays.
  2. Check Each Array One by One:
    • For each array in the input, we look at the current list of common elements (acc) and filter it down to only those elements that also exist in the next array (array).
  3. Filter to Keep Only Common Elements:
    • The filter method checks each element in the current list (acc) to see if it's in the next array (array).
    • If it is, the element stays in the list; if not, it's removed.
  4. Repeat Until All Arrays Are Processed:
    • This process continues until all arrays are processed. The final result is a list of elements that appear in every array.
