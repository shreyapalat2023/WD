Question Answer

1. Difference between “ == “ and “ === “ operators (with examples)

In JavaScript, both the == (loose equality) and === (strict equality) operators are used for comparison, but they behave differently in terms of how they handle types and values.

Loose Equality (==) Operator:

  • The == operator compares the values of two operands after converting them to a common type.
  • It performs type coercion, meaning it may convert the operands to the same type before comparison.
  • If the types of the operands are different, JavaScript attempts to convert them to a common type before making the comparison.
  • This can lead to unexpected results, especially when comparing different data types.
Example 
                console.log(5 == "5"); // Output: true
                console.log(1 == true); // Output: true
                console.log(0 == false); // Output: true

Strict Equality (===) Operator:

  • The === operator compares both the values and the types of two operands.
  • It does not perform type coercion, so the values must be of the same type to be considered equal.
  • If the types of the operands are different, the === operator immediately returns false without attempting to convert them.
Example 
            console.log(5 === "5"); // Output: false
            console.log(1 === true); // Output: false
            console.log(0 === false); // Output: false

JavaScript Comma Operator

Comma Operator (,) mainly evaluates its operands from left to right sequentially and returns the value of the rightmost operand.

When a comma operator is placed in an expression, it executes each expression and returns the rightmost expression.

Example: 
            function Func1() { 
                console.log('one'); 
                return 'one'; 
            } 
            function Func2() { 
                console.log('two'); 
                return 'two'; 
            } 
            function Func3() { 
                console.log('three'); 
                return 'three'; 
            } 
            
            // Three expressions are 
            // given at one place 
            let x = (Func1(), Func2(), Func3()); 
            
            console.log(x);

            Output:
            one
            two
            three
