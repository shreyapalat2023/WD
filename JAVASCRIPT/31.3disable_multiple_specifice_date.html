Disable Multiple Specific Date

Explanation:

  • const disabledDates:This declares a constant variable named disabledDates that stores an array of strings. Each string represents a specific date in the format YYYY-MM-DD that is disabled. Users are not allowed to select these dates.
  • dateInput.addEventListener('input', ...):This line adds an event listener to the dateInput element. It listens for the input event, which is triggered whenever the user changes the value of the input field (i.e., when a date is selected or manually typed).
  • disabledDates.includes(this.value): The includes method checks if the current value of the input field (the selected date) is in the disabledDates array.
    • If this.value matches any date in the disabledDates array, includes returns true; otherwise, it returns false.
